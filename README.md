## <a id="how_to_use">How To Use</a>
```
# clone this repository
$ git clone https://github.com/1021s/Kyle-service.git

# install dependencies
$ npm install

# seed database
$ npm run db:seed

# compile/transpile files with webpack
$ npm run build

# start express server
$ npm start
```
Visit localhost:3001/
